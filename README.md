This program generates a quiz program based on the questions placed in a textfile.
The format of the textfile should be 

question
answer1
answer2
answer3
answer4
correctAnswerIndex
topicstoReview


For example,

// ==========questions.txt====================
Who amogst these was an american president?
Doritos
Monsanto
Obama
Hurricane
2
American Presidents, American Politics
// ===========================================
